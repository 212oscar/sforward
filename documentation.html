Overview

This documentation covers the SF Helper and P4V Helper scripts designed to help mods in the content review, uploading, and building process. These tools help avoid mistakes or misclicks showing the most used information and buttons in an overlay for Salesforce, we only use the information showed on screen this doesn't modify anything on SF just make it easy for you to read the information you need in your workflow.

Before Using:

By using these tools you acknowledge the following:

  • You still need to verify that all the steps in your workflow are being executed correctly.
  • These are just tools to save you some time gathering the needed info.
  • You are still responsible for your own work.
  • In case you found a bug or some tool is not working as expected you should stop using the tool and please report the issue to me (Oscar O. in Slack) and I'll try to fix it.

Features

SFhelper:

Salesforce Tools

Assign to me

With this button you can easily assign a SF case to you, just click it and while is loading you can click Copy SF case and claim the case also in Slack.

Get Info

Will show you a friendly summary of the app names with YES in the colum of "Is New or Changed" from this window you can easily see the supported versions for each app Name copy the app name (to quickly find it in the Depot) or in case you are working on a new submission or Adding a new version (update ADD case) you can click "P4V data" and then with the other tool "P4Vhelper" you can click "Get data" and from there you can Quickly create a folder in the P4V workspace, this will save you a lot of clicks. You can also see in the top of this section the warning color indicator of the seller, Case Type (New submission or Update) and the SF case

We always need the SF case number for everything (claiming the case in slack, notes on p4, etc) thats why I've included a dedicated button for this. Just click it and the case number will be copied to your clipboard.

Add shifts / Show Shifts / Clock-in-out reminders

With this button the script will ask you for your tanda calendar, just go to your Tanda schedule URL (https://my.tanda.co/staff) you must be logged with Okta, click on your name / Attendance points / Schedules / Click on Subscribe iCal / Open that iCal URL, copy all that content, and paste it there.

Now the script will know your shifts and the shift report button will be enabled, if you add more shifts to your schedule then you should "Update shifts" there is a button for that too.

After you saved your calendar you can now enable reminders, so in case you forgot to clock out and you are still in Salesforce working, a small alert will popup with a reminder.

Shift Report

P4Vhelper:

Features

Additional program to help with the workflow of adding new files to the P4V depot, just set your current P4V workspace and the script will help you to create new folders in the depot working seamlessly with the SFhelper, you can fill the needed info by clicking in the SFhelper script the "P4V data" button and then here in the P4Vhelper click on "Get Data", you can also clear workspace directories, and toggle dark mode.

Works better when working with two or more monitors, you can have the Island browser with salesforce in one screen and the P4Vhelper and P4V (Perforce Helix P4V) client in the other.

Installation and basic usage
SFhelper

To install the SFhelper, follow these steps:

  1. Ensure that developer mode is enabled in Island Browser / Manage extensions.
  2. Click on the Tampermonkey icon and select "Dashboard".
  3. Click on the "Utilities" tab on the right side of the dashboard.
  4. In the "Import from URL" section, paste the following URL and click "Import": 
    https://raw.githubusercontent.com/212oscar/sforward/main/tp-uemkp-scripts/SFhelper.js
  5. Confirm the installation of the script.
  6. Make sure Tampermonkey is enabled.
  7. Now it should be working when you enter in Salesforce, if is not showing up, restart the Island Browser.

Alternatively, you can manually install the script:

  1. Open the previous URL in a new tab and copy all the content.
  2. Create a new script and paste the copied info.
  3. Save the script and make sure Tampermonkey is enabled.

P4Vhelper

Doesn't need installation, just follow this simple instructions:

  1. Download the P4Vhelper.exe program from here: Download P4Vhelper.

    2. Alternatively, you can manually install the script:

  2. Save the executable in a custom folder on your computer.
  3. Create a shortcut to the executable on your desktop for easy access.
  4. Open the P4Vhelper application.
  5. Set your current P4V workspace in the application settings.
  6. Use the "Get Data" button in P4Vhelper after clicking "P4V data" in the SFhelper script to fill the needed information.
  7. Use the "Generate" button to quickly create new folders in the depot, you can also clear workspace directories, and toggle dark mode.
  8. Your productivity can be increased a lot if you use two or more monitors: one for the Island browser with Salesforce and the other for the P4Vhelper and P4V (Perforce Helix P4V) client.

Troubleshooting

If you encounter any issues while using the scripts, try the following steps:

  • Ensure Tampermonkey is enabled in your browser.
  • Ensure you have the latest version of the scripts.
  • Some features may not work if Salesforce or Horde page are slow or if they change those pages drastically, but the script can be updated if they change something in Salesforce or Horde.
