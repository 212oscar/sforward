Overview

This documentation covers the SF Helper and P4V Helper scripts designed to help mods in the content review, uploading, and building process. These tools help avoid mistakes or misclicks showing the most used information and buttons in an overlay for Salesforce, this tool only use the information showed on screen this doesn't modify anything on SF just make it easy for you to read the information you need.

Before Using:

By using these tools you acknowledge the following:

  • You still need to verify that all the steps in your workflow are being executed correctly.
  • These are just tools to save you some time gathering the needed info.
  • You are still responsible for your own work.
  • In case you found a bug or some tool is not working as expected you should stop using that tool and report the issue so it can be fixed.

Features
  • Process selected info and copy App Name
  • Copy SF Case
  • Manage cases with floating buttons
  • Send data to P4V Helper to quickly create new folders in the depot
  • Automatically send needed data to the Horde
  • Create new TRC with Product title and Case number as a name
  • Use custom templates (sheets ID) to pre-load a TRC for each product type
  • Organize TRCs into specific folders with GSheet Organizer
  • Easily see your shifts
  • Clock-in / out reminders
  • Easily create a shift report with the cases worked

Installation

To install the scripts, follow these steps:

  1. Open Tampermonkey in your browser.
  2. Create a new script and paste the content of the desired .js file.
  3. Save the script and make sure Tampermonkey is enabled.

Usage

Ensure that you understand the functionality of each script and that it aligns with the project requirements. Use these scripts responsibly to maintain productivity and efficiency.

SF Helper

Overview

The SF Helper script enhances Salesforce functionality, making it easier to access and visualize the specific information needed for the project.

Features

  • One click Copy SF Case
  • Manage cases with floating buttons
  • Show only the needed information (app names with YES) with "Get info" button
  • Create new TRC with Product title and Case number as a name
  • Use custom templates (sheets ID) to pre-load a TRC for each product type
  • Store and see your workforce shifts
  • Clock-in / out reminders
  • Easily create a shift report with the cases worked

Usage

To use the SF Helper script, follow these steps:

  1. Ensure Tampermonkey is enabled in your browser.
  2. Navigate to Salesforce and the script will automatically enhance the functionality.
  3. Use the floating buttons to manage cases and copy information.

P4V Helper

Overview

The P4V Helper script assists in creating new folders in the depot and automating various tasks related to the content review process.

Features

  • Quickly create new folders in the depot
  • Send needed data to the Horde
  • Clear workspace directories
  • Toggle dark mode

Usage

To use the P4V Helper program, follow these steps:

  1. Download the P4V Helper .exe program.
  2. Run the .exe program on your computer.
  3. Select the same P4V workspace used in the P4V visual client.
  4. Use the GUI to select the workspace, enter the required information, and create folders.
  5. Use the "Get Data" button to load data from the clipboard.
  6. Use the "Clear Workspace" button to clear specific subdirectories.
  7. Toggle dark mode using the button at the bottom.

GSheet Organizer

Overview

The GSheet Organizer script helps organize Google Sheets into specific folders and update sheet titles.

Features

  • Organize Google Sheets into specific folders
  • Update sheet titles

Usage

To use the GSheet Organizer script, follow these steps:

  1. Ensure Tampermonkey is enabled in your browser.
  2. Navigate to Google Sheets and the script will automatically add buttons to organize sheets.
  3. Use the buttons to move sheets to specific folders and update titles.

Troubleshooting

If you encounter any issues while using the scripts, try the following steps:

  • Ensure Tampermonkey is enabled in your browser.
  • Check for any errors in the browser console (press F12 to open the console).
  • Ensure you have the latest version of the scripts.
  • Contact the author for further assistance.
